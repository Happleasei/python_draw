## Python绘图总结 代码介绍
```
|- python_draw/
  |- data                           # 数据存储目录
  |- html                           # 生成html文件目录
  |- img                            # 生成图片目录
    |- Advanced.ipynb               # 进阶图像绘画
    |- Basics.ipynb                 # 基本图像绘画
    |- Special.ipynb                # 特殊图像绘画
    |- Study-plotly.ipynb           # plotly工具，绘图可动态交互
```
